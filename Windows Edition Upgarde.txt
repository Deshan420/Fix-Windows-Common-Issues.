Upgrade Windows edition using DISM

Step 1 — Open CMD as Administrator

Step 2 — Check current edition
dism /online /Get-CurrentEdition

Step 3 — Check editions you can upgrade to
dism /online /Get-TargetEditions

If you have any other Editions.
This may show something like: 

Target Edition : Professional

If you already have "Professional Edition" it may show these options..

Target Edition : ProfessionalEducation
Target Edition : ProfessionalWorkstation
Target Edition : Education
Target Edition : ProfessionalCountrySpecific
Target Edition : ProfessionalSingleLanguage
Target Edition : ServerRdsh
Target Edition : IoTEnterprise
Target Edition : Enterprise

Step 4 — Apply the new product key

Replace the example key with yours:

dism /online /Set-Edition:Professional /ProductKey:XXXXX-XXXXX-XXXXX-XXXXX-XXXXX /AcceptEula


Example (Windows 10/11 Pro):

dism /online /Set-Edition:Professional /ProductKey:VK7JG-NPHTM-C97JM-9MPGT-3V66T /AcceptEula


.....Your system will upgrade and reboot automatically.....
